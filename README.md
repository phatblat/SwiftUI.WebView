# WebView

`WKWebView` wrapped in a SwiftUI View. Works on all Apple platforms.

## Requirements

- Xcode 11.4
- Swift 5.2
- iOS 13.0
- macOS 10.15.0
- watchOS 6.0
- tvOS 13.0
